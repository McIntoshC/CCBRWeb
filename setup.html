Setting Up

This step is not necessary for running on Biowulf at NIH, but can provide you with a copy of the Exome Pipeline Software. If you have a Biowulf and Helix account with NIH, you can run it directly on Biowulf. Instructions are provide with the link Run on Biowulf.

git - Using git to clone "ExomePipeline" repository.
Launch Macintosh's Terminal application from the Utility directory. In this example, we will place the repository "ExomePipeline" on the Desktop.

# This is the pretend terminal window
% cd ~/Desktop
% git clone https://github.com/CCBR/ExomePipeline.git
% cd ExomePipeline
% ls
% sh runpipe.sh
